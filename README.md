# Talisman Forge

Talisman Forge is a cross-platform desktop app for Digital Talisman.

## Positioning
A BridgeSpace-inspired workspace, tailored for agency growth ops:
- Prospecting and outreach execution
- Build/delivery coordination
- Daily review and conversion tracking

## Platforms
- macOS 12+
- Windows
- Linux

## Run locally
```bash
cd /home/ubuntu/talisman-forge
npm install
npm run dev
```

## Build installers
```bash
npm run build
```

This generates platform installers via electron-builder.
